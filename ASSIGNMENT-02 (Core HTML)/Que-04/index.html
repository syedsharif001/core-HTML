HTML Tags

  1. DOCTYPE (!): Specifies the document type and version of HTML used in the document.

  2. html: Defines the root element of an HTML page.

  3. head: Contains metadata about the document, such as the title, character set, and linked stylesheets.

  4. title: Sets the title of the web page, which appears in the browser's title bar or tab.

  5. meta: Provides metadata about the document, including character encoding, authorship, and viewport settings.

  6. link: Links external resources, such as stylesheets, to the HTML document.

  7. style: Defines embedded CSS styles for the document.

  8. script: Embeds or references JavaScript code within the HTML document.

  9. Body: Contains the main content of the web page, including text, images, and other elements.

  10. h1 , h2 , h3 , ... h6: Define headings of varying levels, with h1 being the highest and h6 the lowest.
